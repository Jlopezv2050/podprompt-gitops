# PodPrompt - GitOps

Kubernetes manifests and ArgoCD configurations.

## Structure

- `namespaces/`: Namespace definitions
- `argocd/`: ArgoCD applications
- `apps/`: Application manifests (frontend, keycloak, etc)
- `infrastructure/`: Infrastructure components (cert-manager, nginx-ingress)
